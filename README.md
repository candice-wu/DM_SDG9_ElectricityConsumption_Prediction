# 台灣科學園區用電量預測與數據分析平台

本專案旨在建立一個全面的數據分析平台，用於預測台灣各科學園區的用電量，並提供深入的數據洞察。系統利用多種機器學習演算法與數據探勘技術，透過一個互動式的 Streamlit 網站介面，將複雜的數據分析與預測過程視覺化，使其易於理解與操作。

## 專案架構：CRISP-DM 方法論

本專案遵循 **CRISP-DM (跨產業資料探勘標準流程)** 方法論進行開發，確保了從業務理解到最終部署的每個環節都結構清晰、目標明確。整個 Streamlit 應用程式的頁面設計也與 CRISP-DM 的各個階段一一對應：

| CRISP-DM 階段 | 專案對應功能 (Streamlit 頁面) | 說明 |
| :--- | :--- | :--- |
| **1. 業務理解 (Business Understanding)** | `專案整體目標` | 核心目標是預測科學園區的用電量，以輔助相關單位進行能源規劃與決策。 |
| **2. 資料理解 (Data Understanding)** | `📄 資料探索與清理` <br> `📊 資料分析` | 使用者上傳原始資料後，系統會進行初步的探索式資料分析(EDA)，展示資料的統計摘要、分佈與關聯性，以建立對資料的初步認知。 |
| **3. 資料準備 (Data Preparation)** | `📄 資料探索與清理` <br> `♻️ 資料轉換` <br> `🔬 資料精簡` | 此階段涵蓋了資料清理（補值、型態轉換）、特徵工程（自定義排序編碼），以及進階的資料轉換（正規化、離散化、平滑化）與資料精簡技術（維度縮減、數量縮減、資料壓縮），為模型建立打造高品質的資料集。 |
| **4. 建模 (Modeling)** | `🔋 用電量預測` | 提供多種迴歸模型（如線性迴歸、決策樹、梯度提升樹等）供使用者選擇，並可動態調整超參數以進行模型訓練。 |
| **5. 評估 (Evaluation)** | `🔋 用電量預測` <br> `🎭 模型比較分析` | 在預測頁面即時顯示 R², RMSE, MAE 等關鍵效能指標。在模型比較頁面，使用者可以透過視覺化圖表（如預測v.s.實際圖、殘差圖、特徵重要性、混淆矩陣）對不同模型的效能與行為進行深入的比較與評估。 |
| **6. 部署 (Deployment)** | `Streamlit 應用程式` | 整個專案被封裝成一個互動式的 Streamlit 網站，讓非技術背景的使用者也能輕易地操作、探索數據與使用預測模型。 |

## 主要功能亮點

*   **多頁式儀表板**：清晰地分離了資料導入、分析、轉換、預測、評估等功能，提供流暢的使用者體驗。
*   **互動式資料探索**：提供豐富的視覺化圖表（熱圖、迴歸圖、盒鬚圖），並搭配動態生成的數據結論，幫助使用者快速理解數據特性。
*   **全面的數據處理技術展示**：
    *   **資料轉換**：互動式地展示正規化、離散化、資料平滑化的效果。
    *   **資料精簡**：完整演示維度縮減 (PCA, t-SNE)、數量縮減（迴歸、直方圖、叢集、抽樣）及資料壓縮 (DWT, PCA) 等關鍵技術。
*   **彈性的模型訓練與預測**：
    *   支援多種主流迴歸模型（線性迴歸、決策樹、SVR、梯度提升樹等）。
    *   允許使用者即時調整模型超參數。
    *   智慧預設功能：預測未來日期時，系統會自動帶入該地區的歷史同期均溫。
*   **深入的模型評估與比較**：
    *   提供多模型並列的「預測 vs. 實際圖」與「殘差圖」進行效能比較。
    *   視覺化呈現樹模型與線性模型的特徵重要性。
    *   創新的混淆矩陣分析，將連續預測問題轉化為分類評估，以檢驗模型在不同用電量級距的預測準確度。
*   **市場購物籃分析**：內建 Apriori 演算法，使用者可動態選擇特徵、調整支持度與信賴度，以挖掘資料內部有趣的關聯規則。

## 安裝與執行

**1. 環境準備**

建議使用 `conda` 或 `venv` 建立一個獨立的 Python 虛擬環境（建議 Python 3.9 ~ 3.11）。

```bash
# 例如，使用 venv
python -m venv .venv
source .venv/bin/activate  # On Windows, use `.venv\Scripts\activate`
```

**2. 安裝相依套件**

本專案的所有相依套件都記錄在 `requirements.txt` 中。

```bash
pip install -r requirements.txt
```

**3. 範例資料**

本專案已包含 `Final_rawdata_20251022151713.csv` 作為範例資料檔案。您可以在平台啟動後，直接上傳此檔案至「📄 資料探索與清理」頁面，即可體驗所有功能。

**4. 執行應用程式**

```bash
streamlit run 5114050013_DM_SDG9.py
```

執行後，Streamlit 將會在您的終端機提供一個本地網址（通常是 `http://localhost:8501`），請在瀏覽器中開啟以使用本平台。
